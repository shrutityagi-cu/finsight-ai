## Overview 📖 

FinSight AI is an AI-powered investment intelligence platform designed to provide retail investors with institutional-grade insights.

The platform combines:

- Portfolio Analytics 📊 
- Machine Learning 🤖 
- Financial News Intelligence 📰 
- AI Copilot 💬 
- Explainable Investment Predictions 📈 
- Risk & Diversification Analysis 📉 

Instead of only showing numbers, FinSight AI explains **why** an investment decision is recommended.

# Getting Started

Clone the repository

```bash
git clone https://github.com/shrutityagi-cu/finsight-ai.git
```

Install backend dependencies

```bash
cd backend
uv sync
```

Run the server

```bash
uvicorn app.main:app --reload
```
